🥁 JavaScript Drum Kit
Play drum sounds using your keyboard! Press keys A through L to trigger different drum sounds with visual feedback.

🎹 Controls
A - Clap

S - Hi-hat

D - Kick

F - Open Hat

G - Boom

H - Ride

J - Snare

K - Tom

L - Tink

🚀 Quick Start
Clone the repo

Open index.html in your browser

Start drumming!

Tech: HTML, CSS, JavaScript
